# Wordly 📖

A clean, minimal dictionary web app that lets you search for word definitions, phonetics, synonyms, and pronunciations powered by the Free Dictionary API.

**🌐 Live at:** [m-mukundi.github.io/wordly-dictionary](https://m-mukundi.github.io/wordly-dictionary/)

---

## Features

- **Word of the Day** — displays a featured word with its definition and pronunciation on the homepage
- **Search** — look up any English word instantly via the navbar or main search box
- **Definitions** — displays definitions grouped by part of speech
- **Phonetics** — shows the phonetic transcription of the searched word
- **Audio Pronunciation** — play the pronunciation of a word with a single click
- **Synonyms** — browse synonym chips for each word, with click-to-search functionality

---

## How to Use

1. **Search for a word**
   - Type a word into the main search box on the homepage and press **Enter** or click the search button
   - After searching, the navbar search box appears so you can search again without scrolling back up

2. **View a definition**
   - Results show the word, its phonetic pronunciation, part of speech, and definitions
   - Multiple definitions are listed where available

3. **Listen to pronunciation**
   - Click the **play button** next to the word to hear it pronounced
   - The button is hidden automatically if no audio is available for that word

4. **Explore synonyms**
   - Synonyms are displayed as clickable chips below the definitions
   - Click any synonym to search for that word directly
